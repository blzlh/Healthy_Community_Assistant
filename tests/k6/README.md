# k6 压力测试使用指南

## 快速开始

### 1. 安装 k6

**Windows (PowerShell 管理员):**
```powershell
choco install k6
```

或下载安装包：https://github.com/grafana/k6/releases

### 2. 获取测试 Token（可选）

由于项目使用邮箱验证码登录，压力测试需要手动获取 token：

1. 登录应用
2. 按 F12 打开开发者工具
3. 切换到 Application → Local Storage
4. 找到 `token` 或 `supabase-auth-token`
5. 复制 token 值

### 3. 运行测试

```powershell
# 方式一：使用脚本
tests\k6\run-tests.bat api        # API 性能测试
tests\k6\run-tests.bat security   # 安全防护验证
tests\k6\run-tests.bat all        # 运行所有测试

# 方式二：使用环境变量设置 token
set TEST_TOKEN=eyJxxx... && tests\k6\run-tests.bat api

# 方式三：直接使用 k6
k6 run tests/k6/api-performance.js
k6 run tests/k6/security-test.js
```

## 测试说明

| 测试 | 说明 | 是否需要 Token |
|------|------|---------------|
| `api-performance.js` | API 性能测试（P95、QPS） | 可选 |
| `security-test.js` | 安全防护验证 | 可选 |

---

## 测试项目

### API 性能测试

测试内容：
- 公开 API：社区动态列表
- 认证 API：用户资料、健康记录、健康对话

验收指标：
- P95 响应时间 ≤ 500ms
- 错误率 < 5%
- 支持 ≥ 30 并发用户

### 安全防护验证

测试内容：
- **暴力破解检测**：连续发送错误登录请求，验证是否触发封禁
- **接口滥用检测**：高频发送请求，验证是否触发限流

验收指标：
- 暴力破解拦截率 ≥ 95%
- 接口滥用拦截率 ≥ 95%

## 注意事项

1. **测试前确保服务已启动**
   ```powershell
   # 启动后端
   cd apps/backend && pnpm dev
   ```

2. **安全测试会触发封禁**
   - 暴力破解测试可能导致 IP 被临时封禁
   - 封禁时长：10 分钟
   - 可在 Security 页面手动解封

3. **测试报告位置**
   ```
   tests/k6/reports/
   ├── api-performance-report.json
   └── security-test-report.json
   ```
